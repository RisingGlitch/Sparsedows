# Sparsedows
A WORK IN PROGRESS, port of some of Sparsebox's features to python

# How to use:

git pull https://github.com/RisingGlitch/Sparsedows.git
cd Sparsedows

Add your gestalt file into the Sparsedows folder.

python main.py
